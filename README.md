### Prerequisites

In order to run this project you need:

- Ruby 2.7.0
- Rails
- Bundler
- Rails-server
- postgresql

### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone https://github.com/Zeph180/my-budget.git
```

### Usage

To get the project running:


```sh
  cd my-budget
  bundle install
  rails db:create
  rails db:migrate
  rails server
```
